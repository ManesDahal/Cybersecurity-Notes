# Basic Linux Commands

- ls – list files
- cd – change directory
- pwd – show current directory
- cp – copy files
- mv – move files
- rm – remove files

## Why Linux?
Most servers and security tools run on Linux.
# Linux File Permissions

- w = write
- x = execute
